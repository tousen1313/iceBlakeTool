# 🎉 アイスブレイクツール

オンライン会議（Zoom・Teams等）の冒頭を盛り上げる、ランダム質問ジェネレーターです。

## 機能

- ユーモア・軽い系の日本語アイスブレイク質問を1問ずつランダム表示
- 「次の質問へ！」ボタンで即座に次の質問へ切り替え
- 表示した質問の履歴を一覧表示

## 使い方

1. ブラウザでアクセス
2. 画面の質問をオンライン会議参加者に読み上げる（画面共有推奨）
3. 「🎲 次の質問へ！」ボタンで次の質問へ

## 技術スタック

- **フレームワーク**: Next.js 15 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **ホスティング**: Vercel（無料プラン）

## ローカル開発

```bash
npm install
npm run dev
```

`http://localhost:3000` でアクセスできます。

## テスト

```bash
npm test
```

## デプロイ

`main` ブランチへの push で Vercel に自動デプロイされます。
